# CHAT WITH PDF
ChatWithPDF is an AI-powered application that allows users to upload PDF documents and interact with them through natural language conversations, enabling quick information retrieval and document understanding.

This project helps users quickly extract information from large PDFs without manually reading the entire document.

🚀 Features

📂 Upload and analyze PDF documents

💬 Ask questions about the document content

🤖 AI-powered responses using Large Language Models

🔎 Semantic search for accurate information retrieval

⚡ Fast and interactive chat interface

📑 Supports large and multi-page PDFs

🛠️ Tech Stack

Python

LangChain

OpenAI / LLM APIs

FAISS / Vector Database

Streamlit / Flask (UI)

PyPDF / PDF processing libraries
